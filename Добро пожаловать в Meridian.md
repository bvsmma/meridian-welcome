# Добро пожаловать в Meridian

Meridian — это бесплатное приложение для ведения заметок с открытым исходным кодом, мощная альтернатива Obsidian. Создано для писателей, исследователей и разработчиков, которым нужен полный контроль над своими заметками.

---

## Основы

**Пишите** — откройте любой `.md` файл и начните печатать. Meridian отображает предпросмотр в реальном времени.

**Навигация** — используйте `⌘K`, чтобы открыть командную палитру и мгновенно перейти к любому файлу.

**Связывайте заметки** — введите `[[` для создания ссылки на другую заметку. Эти ссылки станут линиями на [[Интерактивный граф]].

**Организуйте** — перетаскивайте файлы на боковой панели, создавайте папки или предоставьте организацию графу.

## Сочетания клавиш

| Действие | Сочетание |
|--------|----------|
| Командная палитра | `⌘K` |
| Новая заметка | `⌘N` |
| Ежедневная заметка | `⌘D` |
| Сохранить | `⌘S` |
| Настройки | `⌘,` |
| Интерактивный граф | `⌘⇧G` |
| Жирный | `⌘B` |
| Курсив | `⌘I` |
| Встроенный код | `⌘\`` |

---

## Готовы начать?

Откройте [[Начало работы]] для первого знакомства или нажмите `⌘⇧G` прямо сейчас, чтобы увидеть, как связаны все заметки в этом хранилище.

> [!TIP] Откройте интерактивный граф
> Нажмите `⌘⇧G`, чтобы увидеть интерактивную карту всех заметок и их связей.
