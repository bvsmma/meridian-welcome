# How to Take Smart Notes

**Author:** Sönke Ahrens
**Year:** 2017
**Rating:** ⭐⭐⭐⭐⭐

---

## Why I read it

I wanted to understand how to make note-taking actually useful — not just storing information, but building knowledge that compounds over time.

---

## The core argument

Most people take notes wrong. They highlight, copy, summarize — but never process. Notes sit in notebooks, never to be seen again.

Ahrens argues for a different system: write notes as if they're permanent, link them to other notes, and let ideas accumulate into a web of knowledge.

> "Writing is not the outcome of thinking; it is the medium in which thinking takes place."

---

## Key ideas

### 1. The slip-box is the thinking partner

Luhmann didn't think *then* write. He thought *through* writing.

### 2. Four types of notes

| Type | Description | Permanence |
|------|-------------|------------|
| Fleeting | Quick captures | Temporary |
| Literature | Notes on sources | Medium |
| Permanent | Fully formed ideas | Permanent |
| Project | Project-specific | Project lifespan |

### 3. Context over categories

Don't ask "where does this go?" Ask "what does this connect to?"

---

## What I'm doing differently

- [ ] Write literature notes in my own words, never copy-paste
- [ ] Write permanent notes the same day I write fleeting notes
- [ ] Link every new permanent note to at least 3 existing notes
- [x] Switched from folder-based to link-based organization
