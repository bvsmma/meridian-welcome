# Tasks & Checklists

Meridian scans every note in your vault for checkboxes and collects them in the Tasks panel — a unified to-do list across all your notes.

---

## Creating tasks

```markdown
- [ ] This is an open task
- [x] This task is done
```

Tasks appear in the Tasks panel (checkbox icon in the sidebar).

---

## Checking off tasks

Click the checkbox in the Tasks panel — the file updates automatically. Or check it directly in the note. Both stay in sync.

---

---

## Filtering tasks

In the Tasks panel filter by **Pending**, **Completed**, or **All**. Use the search bar to find tasks by keyword.

---

## Best practices

**Keep tasks in context** — put tasks in the note they belong to. A task in [[Project Planning]] lives next to the project it's about.

**Use daily notes for today's tasks** — in your [[Daily Notes|daily note]], list what you want to do today. The Tasks panel collects it all.

> [!NOTE]
> Tasks are just text. If you move or rename a note, the task moves with it.
