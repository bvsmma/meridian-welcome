# Search & Command Palette

Two ways to find anything in your vault instantly.

---

## Command Palette — `⌘K`

Press `⌘K` to open it.

- **Search files by name** — start typing a filename
- **Search inside notes** — type `/` followed by your query. Results show a snippet of matching text.
- **Run commands** — type `>` to access app commands

---

## Sidebar search

Click the search icon (magnifying glass) in the left sidebar. Type any word or phrase — Meridian searches all note contents and shows the matching text in context.

---

## Navigation table

| Goal | How |
|------|-----|
| Jump to a note | `⌘K` → type name |
| Search note contents | `⌘K` → `/query` |
| See what links here | Backlinks panel (link icon in sidebar) |
| See all headings | TOC panel (list icon in sidebar) |

---

## Backlinks

The Backlinks panel shows every note that links to the currently open note.

If [[Research Note]] links to this note, you'll see it listed here. This is how you find all context around any idea.

> [!TIP]
> Get comfortable with `⌘K` — it's the most-used shortcut in Meridian.
