# Начало работы

Давайте поможем вам освоиться менее чем за 5 минут.

---

## Шаг 1: Создайте свою первую заметку

Нажмите `⌘N`, чтобы создать новую заметку. Дайте ей имя и начните писать.

Meridian сохраняет изменения автоматически — вы никогда не потеряете черновик.

---

## Шаг 2: Попробуйте Markdown

Meridian использует Markdown для форматирования. Вот всё, что вам нужно знать:

```
**жирный**          →  жирный
*курсив*           →  курсив
~~зачеркнутый~~     →  зачеркнутый
==выделенный==      →  выделенный
`встроенный код`   →  встроенный код
```

Для заголовков начинайте строку с символа `#`:

```
# Заголовок 1
## Заголовок 2
### Заголовок 3
```

Смотрите подробное руководство в [[Руководство по Markdown]].

---

## Шаг 3: Связывайте заметки

Введите две открывающие квадратные скобки в любом месте, чтобы создать ссылку на другую заметку. Эти ссылки превратятся в связи на [[Интерактивный граф]].

---

## Шаг 4: Используйте командную палитру

Нажмите `⌘K`, чтобы открыть командную палитру. С её помощью вы можете:

- Искать любую заметку по названию
- Искать текст внутри заметок с помощью `/`
- Выполнять команды (например, вставлять шаблоны)

---

## Шаг 5: Откройте интерактивный граф

Нажмите `⌘⇧G`, чтобы увидеть интерактивную карту всех заметок и их связей. Подробнее в [[Интерактивный граф]].
