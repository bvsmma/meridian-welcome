# Задачи и списки

Meridian сканирует каждую заметку в вашем хранилище на наличие флажков задач (чекбоксов) и собирает их в единой панели «Задачи» — общем списке дел для всех ваших заметок.

---

## Создание задач

```markdown
- [ ] Это активная задача
- [x] Эта задача выполнена
```

Задачи отображаются на панели задач (иконка галочки на боковой панели).

---

## Отметка о выполнении

Кликните по чекбоксу в панели задач — файл обновится автоматически. Либо отметьте выполнение прямо в тексте заметки. Оба варианта синхронизируются друг с другом.

---

## Примеры задач для этого хранилища

- [ ] Прочитать [[Начало работы]]
- [ ] Открыть [[Интерактивный граф]] и изучить связи
- [ ] Попробовать создать холст в [[Холст]]
- [ ] Написать свою первую ежедневную заметку (`⌘D`)
- [ ] Настроить [[Резервное копирование Git]] для истории версий
- [ ] Создать заметку о том, что вы сейчас изучаете
- [ ] Связать эту заметку как минимум с двумя другими
- [x] Открыть Meridian ✓

---

## Фильтрация задач

В панели задач вы можете фильтровать их по статусу: **Активные**, **Выполненные** или **Все**. Используйте строку поиска для поиска задач по ключевым словам.

---

## Рекомендации

**Держите задачи в контексте** — пишите задачи в тех заметках, к которым они относятся. Задача в заметке [[Планирование проекта]] должна находиться рядом со связанной информацией о проекте.

**Используйте ежедневные заметки для текущих задач** — в своей [[Ежедневные заметки|ежедневной заметке]] перечислите то, что планируете сделать сегодня. Панель задач соберет всё воедино.

> [!NOTE]
> Задачи — это обычный текст. Если вы переместите или переименуете заметку, задача переместится вместе с ней.
