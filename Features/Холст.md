# Холст

Холст — это бесконечное пространственное рабочее место. Размещайте заметки, изображения и текстовые карточки в любом месте свободной доски, а затем соединяйте их линиями.

Откройте [[Обзор Meridian.canvas]], чтобы увидеть готовый пример.

---

## Открытие холста

1. Нажмите иконку холста в нижней панели инструментов на боковой панели.
2. Любой файл с расширением `.canvas` открывается как холст.

---

## Что можно делать

**Добавить карточку** — дважды кликните в любом пустом месте холста, чтобы создать текстовую карточку.

**Встроить заметку** — перетащите `.md` файл с боковой панели на холст. Карточка будет отображать живое содержимое заметки.

**Создать связи** — наведите курсор на границу карточки, чтобы появились точки подключения. Перетащите линию из одной точки к другой карточке.

**Свободное перемещение** — перетаскивайте карточки в любое место. Используйте колесико мыши для масштабирования. Зажмите фон и тяните для перемещения по холсту.

---

## Когда использовать холст

- **Интеллект-карты (Mind maps)** — мозговой штурм темы со всех сторон.
- **Доски проектов** — карточки задач, разложенные по этапам выполнения.
- **Схемы исследований** — размещение источников информации вокруг ключевого вопроса.
- **Диаграммы систем** — отображение связей между компонентами.
