# Добро пожаловать в Meridian

Meridian — локальное приложение для заметок в Markdown. Оно хранит данные в обычных файлах, показывает связи между заметками и помогает держать рабочие материалы под контролем.

## Базовый поток

**Пишите** — откройте любой `.md` файл и начните редактировать.

**Переходите быстро** — `Ctrl+K` открывает палитру команд и поиск по файлам.

**Связывайте заметки** — введите `[[`, чтобы создать ссылку на другую заметку.

**Смотрите структуру** — граф показывает, как ваши идеи соединяются между собой.
