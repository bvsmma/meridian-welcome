# Getting Started

Let's get you up and running in under 5 minutes.

---

## Step 1: Create your first note

Press `Ctrl+N` to create a new note. Give it a name and start typing.

Meridian saves automatically — you'll never lose a draft.

---

## Step 2: Try Markdown

Meridian uses Markdown for formatting. Here's everything you need:

```
**bold**          →  bold
*italic*          →  italic
~~strikethrough~~ →  strikethrough
==highlight==     →  highlight
`inline code`     →  inline code
```

For headings, start a line with `#`:

```
# Heading 1
## Heading 2
### Heading 3
```

See the full reference in [[Markdown Guide]].

---

## Step 3: Link your notes

Type two opening square brackets anywhere to link to another note. These links become connections in the [[Graph View]].

---

## Step 4: Use the Command Palette

Press `Ctrl+K` to open the Command Palette. From there you can:

- Search any note by name
- Search inside notes with `/`
- Run commands like creating templates

---

## Step 5: Open the Graph View

Press `Ctrl+Shift+G` to see a live visualization of all your notes and how they connect. See [[Graph View]] for details.
