# Welcome to Meridian

Meridian is a free, open-source note-taking app — a powerful alternative to Obsidian. Built for writers, researchers, and developers who want full control over their notes.

## The basics

**Write** — open any `.md` file and start typing. Meridian renders a live preview on the right.

**Navigate** — use `Ctrl+K` to open the Command Palette and jump to any file instantly.

**Link notes** — type `[[` to link to another note. These links become edges in the [[Graph View]].

**Organize** — drag files in the sidebar, create folders, or let the graph do the organizing for you.
