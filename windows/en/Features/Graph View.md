# Graph View

The Graph View turns your vault into a living map of ideas. Every note is a node. Every `[[link]]` is an edge.

Press `Ctrl+Shift+G` to open it.

---

## What you're seeing

Each dot is a note. Lines between dots are wiki-links. The more links a note has, the larger its dot.

This note links to: [[Welcome to Meridian]], [[Getting Started]], [[Markdown Guide]], [[Canvas]], [[Research Note]].

Open the graph now and watch this note light up as a hub.

---

## How to use it

**Zoom** — scroll to zoom in/out. Pinch on trackpad.

**Pan** — click and drag the background.

**Click a node** — opens that note in the editor.

---

## Why it matters

A flat list of files hides the structure of your thinking. The graph reveals it.

Look for:
- **Hubs** — notes that many others link to. These are your key concepts.
- **Clusters** — groups of tightly connected notes. These are topic areas.
- **Islands** — isolated notes with no links. These might need connecting.

---

## Building a useful graph

1. **Use wiki-links freely** — link concepts, not just sources
2. **Create index notes** — one note per topic that links to all related notes
3. **Link bidirectionally** — if A links to B, consider linking B back to A

See [[Research Note]] for an example of a well-linked note.

> [!TIP]
> The graph updates in real time as you create and link notes.
